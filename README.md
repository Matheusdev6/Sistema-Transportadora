# Sistema de Gerenciamento de Veículos

Projeto desenvolvido em Java para praticar conceitos de Orientação a Objetos (POO).

## Tecnologias e Conceitos
- **Linguagem:** Java
- **Conceitos de POO:** Herança, Polimorfismo e Encapsulamento.

## Estrutura do Projeto
- `src/`: Contém o código-fonte.
- `Main.java`: Classe principal para execução do sistema.

## Como executar
1. Clone o repositório.
2. Navegue até a pasta `src`.
3. Compile: `javac Main.java`
4. Execute: `java Main`
