# Sistema-Transportadora
Sistema feito em Java focado para o aprendizado da linguagem, em especial para os conceitos de Programação Orientada a Objetos, Encapsulamento, e lógica de programação. 
O programa solicita ao usuário que digite a cidade de saída, de chegada, distância, quantidade de pedágios. O software analisa qual o modal de transporte mais adequado considerando apenas 
distância.
